ROBÓTICA

1. Introdução à Robótica e Sistemas Embarcados

Definição e Aplicações

Robótica é a área que combina eletrônica, mecânica e programação para criar máquinas que realizam tarefas de forma autônoma ou semi-autônoma. Exemplos: braços robóticos em fábricas, drones, robôs de resgate.

Sistemas embarcados são dispositivos projetados para executar funções específicas, como controladores em carros, termostatos e relógios digitais.

Conexão entre Hardware e Software

O hardware (sensores, atuadores) capta informações ou realiza ações no ambiente físico. O software (código) define como o hardware deve se comportar.

Exemplos práticos:
Um sensor de temperatura envia dados para o Arduino, e o software decide quando ligar um ventilador.

2. O que é o Arduino?

Plataforma de Protótipos

Arduino é uma ferramenta poderosa para aprender eletrônica e programação. Exemplo de aplicação: Controlar LEDs, motores e sensores para criar projetos interativos.

Componentes do Arduino Uno

Pinos Digitais (D0-D13): Entrada/saída para sinais digitais (ex.: ligar/desligar LEDs).
Pinos Analógicos (A0-A5): Leitura de valores analógicos, como sensores de temperatura.
Microcontrolador: Onde o código é executado.
Fonte de Alimentação: Conexão por USB ou pino de alimentação externa.

3. Conceitos Básicos de Eletrônica

Componentes Comuns

LED: Emite luz ao ser energizado. Resistor: Controla a corrente elétrica para proteger componentes. Sensor Ultrassônico: Mede distância através de ondas de som. Servo Motor: Gira para posições específicas baseado em sinais de controle.

Exemplo Prático no Tinkercad: Acender um LED com um Resistor

  • 1. Acesse e crie um novo circuito.
  • 2. Adicione um Arduino Uno, um LED e um resistor de 220 ohms.
  • 3. Conecte:
    • O ânodo (perna longa) do LED ao pino 13 do Arduino.
    • O cátodo ao resistor e depois ao GND.

Código no Arduino IDE:

void setup() {
    pinMode(13, OUTPUT);
}

void loop() {
    digitalWrite(13, HIGH); // Liga o LED
    delay(1000);            // Espera 1 segundo
    digitalWrite(13, LOW);  // Desliga o LED
    delay(1000);            // Espera 1 segundo
}

5. Montando Circuitos Simples

Exemplo 1: Controle de Brilho do LED com Potenciômetro

  1. 1. Conecte um potenciômetro aos pinos VCC, GND e A0 do Arduino.
  2. 2. Ligue um LED ao pino 9 com um resistor.

Código:

    
    int potPin = A0;      // Pino do potenciômetro
    int ledPin = 9;       // Pino do LED
    int value;            // Armazena o valor lido
    
    void setup() {
        pinMode(ledPin, OUTPUT);
    }
    
    void loop() {
        value = analogRead(potPin);       // Lê o valor do potenciômetro
        analogWrite(ledPin, value / 4);  // Ajusta o brilho (0-255)
    }

6. Programação com Arduino IDE

Estrutura do código

  • 1. setup(): Configurações iniciais, como definir pinos de entrada ou saída.
  • 2. loop(): Executa repetidamente enquanto o Arduino está ligado.

Dicas

  • . Use comentários (//) para descrever o que o código faz.
  • . Teste o código em partes pequenas para depurar erros mais facilmente.

Projeto 2 - Controle de Servo Motor com Potenciômetro

Este projeto conecta um potenciômetro ao Arduino para controlar o ângulo de um servo motor.

Componentes Necessários:

  • 1 Servo Motor
  • 1 Potenciômetro
  • Fios de conexão
  • Arduino Uno

Circuito no Tinkercad:

  1. Conecte o potenciômetro:

    • VCC no pino positivo do potenciômetro.
    • GND no pino negativo.
    • A saída do meio no pino A0 do Arduino.

  2. Conecte o servo motor:

    • O fio de controle (geralmente laranja) ao pino 9.
    • O fio de energia (vermelho) ao VCC.
    • O fio GND (marrom) ao GND do Arduino.

Projeto 3 - Robô Seguidor de Linha

Objetivo: Criar um robô que siga uma linha preta em um fundo branco usando sensores infravermelhos.

Componentes Necessários:

  • 2 Sensores infravermelhos (IR)
  • 2 Motores DC
  • 1 Driver de motor L298N
  • Arduino Uno

Circuito no Tinkercad:

  1. Conecte os sensores IR aos pinos digitais (ex.: 2 e 3).

  2. Conecte os motores ao driver L298N:

    • Motor esquerdo aos pinos OUT1 e OUT2.
    • Motor direito aos pinos OUT3 e OUT4.

  3. Conecte o driver ao Arduino:

    • ENA (esquerdo) no pino 10.
    • ENB (direito) no pino 11.

Código no Arduino IDE:

 int leftSensor = 2; // Sensor esquerdo 

            int rightSensor = 3; // Sensor direito 

            int motorLeft = 10; // Motor esquerdo 

            int motorRight = 11; // Motor direito 

            void setup() { 

                pinMode(leftSensor, INPUT);

                 pinMode(rightSensor, INPUT);

                  pinMode(motorLeft, OUTPUT); 

                  pinMode(motorRight, OUTPUT); 

                } 

                void loop() { 

                    int leftValue = digitalRead(leftSensor); // Lê sensor esquerdo 

                    int rightValue = digitalRead(rightSensor); // Lê sensor direito 

                    if (leftValue == LOW && rightValue == HIGH) { 

                        // Se apenas o sensor esquerdo detectar linha 

                        digitalWrite(motorLeft, LOW); 

                        digitalWrite(motorRight, HIGH); 

                    } 

                    else if (leftValue == HIGH && rightValue == LOW) { 

                        // Se apenas o sensor direito detectar linha 

                        digitalWrite(motorLeft, HIGH); 

                        digitalWrite(motorRight, LOW); 

                    }

                     else { 

                        // Se ambos detectarem linha ou nenhum detectar 

                        digitalWrite(motorLeft, HIGH); 

                        digitalWrite(motorRight, HIGH); 

                    }

                 }

Explicação:


  • Sensores infravermelhos detectam a linha preta com menor reflexão de luz.

  • O robô ajusta os motores para seguir a linha detectada.
