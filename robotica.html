ROBÓTICA

1. Introdução à Robótica e Sistemas Embarcados

Definição e Aplicações

Robótica é a área que combina eletrônica, mecânica e programação para criar máquinas que realizam tarefas de forma autônoma ou semi-autônoma. Exemplos: braços robóticos em fábricas, drones, robôs de resgate.

Sistemas embarcados são dispositivos projetados para executar funções específicas, como controladores em carros, termostatos e relógios digitais.

Conexão entre Hardware e Software

O hardware (sensores, atuadores) capta informações ou realiza ações no ambiente físico. O software (código) define como o hardware deve se comportar.

Exemplos práticos:
Um sensor de temperatura envia dados para o Arduino, e o software decide quando ligar um ventilador.
