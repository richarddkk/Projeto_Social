    OFICINAS

    Guia do mochileiro

    É o processo de escrever códigos para criar aplicativos, sites, jogos e sistemas. Há diferentes linguagens, cada uma adequada para certos objetivos, como Python (geral), JavaScript (web), ou C++ (robótica e sistemas).

    A Jornada Conectada

    Essa área explora como os dispositivos se conectam para compartilhar informações. Vai desde configurar redes Wi-Fi até entender os princípios de segurança e comunicação entre sistemas.

    Caminho do Designer

    O UX/UI Design trabalha na criação de interfaces e experiências para o usuário. Envolve tanto o planejamento de como algo funciona (UX) quanto o design visual de como isso é apresentado (UI).

    Mestres das Máquinas

    Robótica combina mecânica, eletrônica e programação para criar máquinas que automatizam tarefas ou ajudam os humanos.
